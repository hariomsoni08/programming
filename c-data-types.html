C Data Types

1. Basic Data Types

a. int

Definition: Represents integer values (whole numbers).

Size: Typically 4 bytes (32 bits).

Range: Signed: -2,147,483,648 to 2,147,483,647; Unsigned: 0 to 4,294,967,295.

Syntax:

int num = 10; // Declaring an integer variable

b. float

Definition: Represents single-precision floating-point numbers.

Size: Typically 4 bytes (32 bits).

Range: Approximately -3.4E+38 to 3.4E+38.

Syntax:

float price = 19.99f; // Declaring a float variable

c. double

Definition: Represents double-precision floating-point numbers.

Size: Typically 8 bytes (64 bits).

Range: Approximately -1.7E+308 to 1.7E+308.

Syntax:

double pi = 3.14159; // Declaring a double variable

d. char

Definition: Represents single characters.

Size: Typically 1 byte (8 bits).

Range: Signed: -128 to 127; Unsigned: 0 to 255.

Syntax:

char letter = 'A'; // Declaring a char variable

2. Derived Data Types

a. Array

Definition: A collection of elements of the same type, stored in contiguous memory locations.

Syntax:

int numbers[5] = {1, 2, 3, 4, 5}; // Declaring an array of integers

b. Structure

Definition: A user-defined data type that allows grouping variables of different types.

Syntax:

struct Person {
    char name[50];
    int age;
};

struct Person p1 = {"Alice", 30}; // Creating an instance of the structure

c. Union

Definition: A user-defined data type that allows storing different data types in the same memory location, but only one at a time.

Syntax:

union Data {
    int intVal;
    float floatVal;
    char charVal;
};

union Data data; // Declaring a union
data.intVal = 10; // Only one member can hold a value at a time

d. Enumeration

Definition: A user-defined data type that consists of integral constants, allowing for named constants.

Syntax:

enum Weekday {Sunday, Monday, Tuesday, Wednesday, Thursday, Friday, Saturday};

enum Weekday today = Wednesday; // Using enumeration

3. Void Data Type

Definition: Represents the absence of a value. It is used for functions that do not return a value.

Syntax:

void functionName() {
    // No return value
}

4. Modifiers

a. signed

Allows for both negative and positive values.

Syntax:

signed int num = -10; // Signed integer

b. unsigned

Only allows for positive values.

Syntax:

unsigned int num = 10; // Unsigned integer

c. short

Represents a short integer (usually 2 bytes).

Syntax:

short int num = 1000; // Short integer

d. long

Represents a long integer (usually 4 bytes or more).

Syntax:

long int num = 1000000; // Long integer

e. long long

Represents a long long integer (typically 8 bytes).

Syntax:

long long int num = 123456789012345; // Long long integer

5. Example Program

Hereâ€™s a simple program demonstrating the usage of various data types:

#include <stdio.h>

// Structure definition
struct Person {
    char name[50];
    int age;
};

int main() {
    // Basic data types
    int age = 25;
    float height = 5.9f;
    double salary = 75000.50;
    char initial = 'J';

    // Structure usage
    struct Person person1;
    person1.age = age;
    snprintf(person1.name, sizeof(person1.name), "John Doe");

    // Output
    printf("Name: %s\n", person1.name);
    printf("Age: %d\n", person1.age);
    printf("Height: %.2f\n", height);
    printf("Salary: %.2lf\n", salary);
    printf("Initial: %c\n", initial);

    // Array example
    int numbers[5] = {1, 2, 3, 4, 5};
    printf("Numbers: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");

    return 0;
}
