Introduction to C Programming

1. Introduction to Programming

Programming is the process of designing and building executable computer software to accomplish a specific task...

2. What is a Programming Language?

A programming language is a formal language comprising a set of instructions that can be used to produce various kinds of output...

3. What is Coding?

Coding, or programming, is the act of writing code in a programming language to create software programs...

4. What is C?

C is a high-level programming language that was developed in the early 1970s by Dennis Ritchie at Bell Labs...

5. Applications of C Language

  • System Programming: C is used to develop system software...
  • Embedded Systems: It is extensively used in embedded systems...
  • Game Development: Many games are developed using C/C++ for performance...
  • Database Management: C is used to implement database systems...
  • Graphics and GUI: C provides libraries for graphics programming...

6. Why Learn C?

  • Foundation for Other Languages: C is the basis for many modern programming languages...
  • Efficiency: C programs are often faster and more efficient...
  • Control: C allows low-level access to memory and hardware...
  • Portability: C programs can be compiled and run on various platforms...

7. Key Points About C

  • Structured Language: C follows a structured programming paradigm...
  • Case-Sensitive: C distinguishes between uppercase and lowercase letters...
  • Rich Library: C has a rich set of built-in functions and libraries...
  • Modularity: C allows the division of programs into functions...

8. First C Program

#include <stdio.h>

int main() {
    printf("Hello, World!\\n");
    return 0;
}

Explanation:

  • #include <stdio.h>: This line includes the standard input-output library...
  • int main(): This is the main function where the program execution begins...
  • printf("Hello, World!\\n");: This line prints "Hello, World!" to the console...
  • return 0;: This statement indicates that the program ended successfully...

9. Basic Syntax of C

Comments: C supports single-line (//) and multi-line (/* ... */) comments...

A variable is a named storage location in memory...

Example of Variable Declaration:

int age; // Declaration of an integer variable
float salary; // Declaration of a float variable
char initial; // Declaration of a character variable

10. Data Types in C

C supports several data types:

  • int: Integer type (e.g., int a = 10;)
  • float: Floating-point type (e.g., float b = 5.5;)
  • double: Double precision floating-point (e.g., double c = 10.75;)
  • char: Character type (e.g., char d = 'A';)

11. Control Flow Statements

C provides various control flow statements to manage the execution of code blocks...

If-Else Statement:

if (condition) {
    // code to execute if condition is true
} else {
    // code to execute if condition is false
}

Switch Statement:

switch (variable) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // default code
}

Loops:

For Loop:

for (int i = 0; i < 10; i++) {
    printf("%d\\n", i);
}

While Loop:

int i = 0;
while (i < 10) {
    printf("%d\\n", i);
    i++;
}

12. Functions in C

Functions are a block of code that performs a specific task and can be reused...

Example of Function Declaration and Definition:

// Function declaration
int add(int a, int b);

int main() {
    int result = add(5, 10);
    printf("Sum: %d\\n", result);
    return 0;
}

// Function definition
int add(int a, int b) {
    return a + b;
}

Conclusion

C is a powerful programming language that forms the basis for many modern languages and applications...
